# Pystagram Project
## 1. 기능 설정
- 인증 시스템: 사용자 등록, 로그인, 로그아웃
- 피드 페이지: 사용자 피드 제공
- 글과 댓글: 사용자가 글을 작성하고 댓글을 달 수 있음
- 동적 URL: 사용자가 작성한 글 및 페이지 동적 URL 처리
- 해시 태그: 해시 태그를 이용한 글 분류
- 글 상세 페이지: 개별 글에 대한 상세 페이지 제공
- 좋아요 기능: 사용자가 글에 좋아요를 누를 수 있음
- 팔로우/팔로잉 기능: 사용자가 다른 사용자와 팔로우 관계 설정

## 2. 환경 설정
- 가상환경 생성 및 활성화
